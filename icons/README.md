# TruthSeek Icons

This directory contains the extension icon assets.

## Required Files

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Design Guidelines

- **Color Scheme**: Purple gradient (#667eea to #764ba2) matching the extension theme
- **Symbol**: Consider a checkmark, magnifying glass, or shield symbol
- **Style**: Modern, flat design with good contrast
- **Format**: PNG with transparency

## Temporary Placeholder

For development, you can create simple colored squares:
- Use any image editor to create 16x16, 48x48, and 128x128 PNG files
- Fill with the brand purple color (#667eea)
- Add a white checkmark or "T" symbol

## Production Icons

Before publishing, replace these placeholders with professionally designed icons that:
- Are recognizable at small sizes
- Convey the "fact-checking" purpose
- Match the extension's visual identity
- Meet Chrome Web Store guidelines
